1. Create Date Table 

Go to data view after loading the data.

Click on new table tab on the ribbon.

Copy and paste the code below in the code editor:


 Date =

ADDCOLUMNS (

CALENDAR (DATE(2014,1,1), DATE(2017,12,31)),

"DateAsInteger", FORMAT ( [Date], "YYYYMMDD" ),

"Year", YEAR ( [Date] ),

"Monthnumber", FORMAT ( [Date], "MM" ),

"YearMonthnumber", FORMAT ( [Date], "YYYY/MM" ),

"YearMonthShort", FORMAT ( [Date], "YYYY/mmm" ),

"MonthNameShort", FORMAT ( [Date], "mmm" ),

"MonthNameLong", FORMAT ( [Date], "mmmm" ),

"DayOfWeekNumber", WEEKDAY ( [Date] ),

"DayOfWeek", FORMAT ( [Date], "dddd" ),

"DayOfWeekShort", FORMAT ( [Date], "ddd" ),

"Quarter", "Q" & FORMAT ( [Date], "Q" ),

"YearQuarter", FORMAT ( [Date], "YYYY" ) & "/Q" & FORMAT ( [Date], "Q" )

)


Right click on the date table and select mark as date.

After creating measure table create the following individual measures in the table

 

Before creating the measures rename the 'loan amount' column to 'Requested Loan_amount'

1. Total Loans_Funded = Sum (Kiva Loans [funded_amount])


2. Total Loans_Requested = SUM ('Kiva Loans'[Requested Loan amount])

3.  %Funded = IF(DIVIDE([Total Loans Funded],[Total Loans Requested]) =0,BLANK(), DIVIDE([Total Loans_Funded],[Total Loans_Requested]))+0

4. YOY LY = CALCULATE([.Total Loans_Funded],DATEADD('Date'[Date],-1,YEAR) )

5. YOY02% = SWITCH ( TRUE (), [YOY02] < 0, UNICHAR ( 9660 ), UNICHAR ( 9650 ) )
    & ROUND ( [YOY02], 4 ) * 100 & " %"
 

6. YOY02 = CALCULATE('Kiva Measures'[Total Loans_Funded] / 'Kiva Measures'[.YOY LY])-1
