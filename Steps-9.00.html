Import Data from Excel source 9

1. Open the zip file (Use the red icon/button above) and extract all the files, select the  excel  file for Kiva  loans.

2. Save it at a location in your computer.

3. Open the Power BI desktop, go to get data and select excel as the data source.
4. In the pop up page, browse to the location of the file.
5. Select the filename.
 6. At the bottom right side of the screen, select load and the data will be loaded to the query editor.
