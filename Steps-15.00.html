Create Line and clustered Column chart for funded loans by Interval 15

While creating the line and clustered column chart, choose the visual then drag the month under date hierarchy column to the shared axis, drag the %funded loans measure to the column values and the YOY measure on the line values.

Then go to the format icon and customize your visual with the steps indicated below.

X: 736

Y: 566

W: 859

H: 442

Legend - on, X axis - on,   Y axis -on,    data labels - on,   Title -on,   Title alignment - center, Word wrap -ON,  text size -18,  Font family - Segoe Bold, tooltip -on,    YOY02 data color - #a8337d,    default data color - #70bbff, Y-axis color - #252423

Title - on, Title background -#0d6abf,   title font - #ffffff,  Title text size -18, A  category -off, word wrap -on, border on



On the Y axis set this up ; Text size -14

To set secondary axis  scroll down on the Y-axis  and turn  the Show secondary  -ON , Set secondary axis color - #a8337d
